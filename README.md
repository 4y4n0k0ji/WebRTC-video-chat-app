## WebRTC Video Chat App

## Description

Une application de chat vidéo React utilisant WebRTC.

## Les commandes

#### Pour installer les dépendances dans le côté client et côté serveur :

```
npm install
```

#### Pour démarrer le serveur (api) :

```
npm start
```

ou

```
nodemon index.js
```

#### Pour démarrer le serveur (client) :

```
npm start
```
